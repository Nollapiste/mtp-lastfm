h2. MTP-Lastfm

The purpose of this program is to scrobble tracks from mtp devices
(such as the Creative Zen, or the Zune) to last.fm. 

h3. Dependencies:
* Python
* python-sqlite
* sqlite3
* mtp-tools
* py-gtk
* gtk
* gtk-glade
* python-elementtree 

These should be installed by default, apart from mtp-tools and python-elementtree

To run: python mtp-lastfm-gtk.py

You can track the projects milestones and submit bug/feature requests at "Lighthouse":http://wodemoneke.lighthouseapp.com/projects/26145-mtp-lastfm/overview.

At the moment, sending love to last.fm is not fully setup, this will be implemented in the next version.
